# Getting Started

## Prerequisites

- [Node.js](https://nodejs.org/en) v20
- [npm](https://www.npmjs.com/) 9.6.4

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Marc9K/nc-news.git
   ```
1. Install dependencies
   ```sh
   npm install
   ```
1. Run locally
   ```sh
   npm run dev
   ```
